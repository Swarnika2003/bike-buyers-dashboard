#Bike Buyers Data Analysis (Interactive Dashboard using MS Excel)
##Project Objective
The objective of this project is to analyze customer demographics and purchasing behavior to understand the factors influencing bike purchases. This analysis helps identify customer trends and business insights using interactive Excel dashboards.

##Dataset Used
- <a href="https://github.com/Swarnika2003/bike-buyers-dashboard/blob/main/Excel%20Project%20Dataset.xlsx"> Excel Project Dataset</a>

##Questions (KPIs)
What is the relationship between income and bike purchases?
Which age group purchases bikes the most?
How does commute distance affect bike purchases?
Compare bike purchases based on gender.
Which region has the highest number of bike buyers?
Compare bike purchases based on marital status.
Which occupation category purchases more bikes?
Analyze education level vs bike purchases.

##Process
Imported and reviewed the Excel project dataset in MS Excel.
Removed duplicate records to improve data accuracy and consistency.
Applied filters and used the Text Contains rule to organize and analyze specific data fields.
Checked and cleaned data for formatting issues, missing values, and inconsistencies.
Created multiple Pivot Tables based on key business questions and KPIs.
Developed Pivot Charts to visualize customer purchasing behavior and sales trends.
Combined Pivot Tables and charts into a single interactive dashboard.
Inserted slicers to enable dynamic filtering by categories such as gender, region, marital status, and occupation.
Designed the dashboard layout for clear visualization and easy user interaction.
